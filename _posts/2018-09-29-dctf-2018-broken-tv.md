The challenge started with this simple website:

![alt text](/uploads/Screenshot%20from%202018-09-29%2021-16-00.png)

We can observe that the following image is changed at about 1 minute:

![alt text](/uploads/blur.png)

Using the following Python script we can extract only the scren image with the shifted pixels.

```py
from PIL import Image 
from PIL import ImageFilter

cropped.save("e.png")
```
